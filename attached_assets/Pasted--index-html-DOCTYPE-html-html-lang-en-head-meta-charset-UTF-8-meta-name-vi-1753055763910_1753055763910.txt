# My Name Is — Learn Your Name with Voice & Photo

A simple, fun React app for toddlers to learn their names — letter by letter — using their own photo and their parent's voice.

## 🌟 Features
- Upload your child’s photo (stored locally only)
- Record each letter's sound using your voice
- Add a bedtime sentence (default: “Divine, time for bed!”)
- Optional coming soon: Sing your child’s name, Make a rhyme
- Flashcard mode with auto-play audio and swipe navigation
- 100% offline. No server. Privacy-first.

## 🧠 Philosophy
Designed for joyful bonding and early phonics. You are the teacher. Your child is the star.

## 🚀 Quick Start
1. Clone or paste into [Replit](https://replit.com)
2. Run `npm install`
3. Run `npm run dev`

## 🛠 Tech Stack
- React 18
- TailwindCSS
- Lucide Icons
- Vite

## 👶 Accessibility
- Visuals optimized for toddlers
- Full keyboard navigation
- Deaf-friendly mode (in roadmap)
